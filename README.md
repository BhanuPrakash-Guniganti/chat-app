

# Chat Application

An elegant, beginner-friendly chat app with stunning glassmorphism UI built using HTML, CSS, and JavaScript. The interface features modern gradients, frosted glass effects, and smooth animations—offering a luxury web experience.

## Features

- Beautiful glassmorphism main panel and chat bubbles
- Responsive layout for desktop and mobile devices
- Gradient backgrounds and premium button styling
- Message animations for smooth appearance
- Easy to use and extend for beginners


### Prerequisites

- [VS Code]
- [Live Server Extension]

### Installation

1. **Clone or Download** this repository.
2. Open the project folder in VS Code.
3. Install the "Live Server" extension in VS Code.
4. Right-click `index.html` and select “Open with Live Server”.

### Project Structure

```
premium-chat-app/
├── index.html
├── style.css
├── app.js
```


## Customization

- Tweak color schemes or gradients in `style.css` for your own branding.
- Extend message handling in `app.js` for new features (emojis, avatars, etc).

## Technologies Used

- HTML5
- CSS3 (Glassmorphism, gradients, transitions)
- JavaScript (Vanilla)
